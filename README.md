# Tower-Defence-C-Project

## Introduction
This project is a simple console-based Tower Defence game implemented
using C programming.

## Objective
To stop enemies from reaching the endpoint using defensive towers.

## Features
- Enemy movement
- Tower attack system
- Health reduction logic
- Console-based gameplay

## How to Run
1. Compile:
   gcc tower_defence.c -o tower
2. Run:
   ./tower
